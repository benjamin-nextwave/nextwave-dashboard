# Requirements: NextWave Client Management Dashboard

**Defined:** 2026-02-07
**Core Value:** Every task deadline stays current -- no task ever shows a past date. The daily dashboard always reflects what needs attention today.

## v1 Requirements

### Foundation

- [x] **FOUN-01**: Supabase database with `companies` and `tasks` tables, proper types, FK relationship, cascade deletes
- [x] **FOUN-02**: Supabase singleton client configured via environment variables (NEXT_PUBLIC_SUPABASE_URL, NEXT_PUBLIC_SUPABASE_ANON_KEY)
- [x] **FOUN-03**: Centralized date utility module with Europe/Amsterdam timezone, Dutch locale, Monday week start
- [x] **FOUN-04**: Overdue-rolling computation: effective_deadline = max(deadline, today), days_overdue = dateDiff(today, original_deadline) when overdue and not completed
- [x] **FOUN-05**: Dark/light mode toggle persisted in localStorage with proper hydration (no flash)
- [x] **FOUN-06**: Navigation with 3 items (Dashboard, Gantt, Bedrijven), dark mode toggle, active page indicator

### Default Tasks

- [x] **DTSK-01**: When a company is created, 6 default tasks are auto-generated starting from warmup_start_date, each 1 day apart
- [x] **DTSK-02**: Default task 1 (day 1): "Create email templates" -- date not editable
- [x] **DTSK-03**: Default task 2 (day 2): "Have an onboarding call" -- date IS editable
- [x] **DTSK-04**: Default task 3 (day 3): "Buy custom domains and mailboxes, place in warmup tool" -- date not editable
- [x] **DTSK-05**: Default task 4 (day 4): "Create an RLM" -- date not editable
- [x] **DTSK-06**: Default task 5 (day 5): "Create follow-up mails" -- date not editable
- [x] **DTSK-07**: Default task 6 (day 6): "Create a dashboard" -- date not editable
- [x] **DTSK-08**: All default tasks can be marked as completed but only "onboarding call" has an editable date

### Homepage

- [x] **HOME-01**: Header showing "Vandaag" with current date formatted in Dutch (e.g., "Maandag 10 februari 2025")
- [x] **HOME-02**: Progress donut chart showing completed/total task ratio with color gradient (green when done, red/orange for incomplete) and "3/7" counter in center
- [x] **HOME-03**: Task list showing all tasks with deadline = today (including rolled-over overdue tasks)
- [x] **HOME-04**: Each task shows company name, task title, and overdue badge (-1, -2 etc in red) if applicable
- [x] **HOME-05**: Completed tasks shown with strikethrough and green checkmark
- [x] **HOME-06**: Tasks sorted: urgent first, then by overdue days (most overdue first), then alphabetical
- [x] **HOME-07**: Clicking a task opens the same edit overlay as in Gantt view

### Gantt Chart

- [x] **GANT-01**: Left sidebar with company names, right area with horizontally scrollable timeline grid (days as columns)
- [x] **GANT-02**: Each company has a light blue semi-transparent bar spanning from warmup_start_date to go_live_date
- [x] **GANT-03**: Tasks appear as colored markers on their deadline date within the company's row (red = pending, green = completed)
- [x] **GANT-04**: Overdue tasks show -N delay badge, urgent tasks have distinct visual indicator
- [x] **GANT-05**: Click task marker opens edit overlay with: editable title, urgency toggle, notes textarea, completion checkbox, save/cancel, delete with confirmation
- [x] **GANT-06**: Click company name opens overlay showing all tasks for that company sorted by deadline ascending, each clickable to open task edit overlay
- [x] **GANT-07**: "+ Nieuwe taak" button per company to create tasks (title + deadline fields, company pre-selected)
- [x] **GANT-08**: Default view shows current week + next week (14 days), today's column highlighted
- [x] **GANT-09**: Arrow buttons and horizontal scroll for week navigation, "Vandaag" button to jump to current view

### Company Management

- [x] **COMP-01**: Overview page with grid of company cards showing name, go-live date, number of open tasks
- [x] **COMP-02**: "+ Nieuw bedrijf" button opening form with name, warmup start date, go-live date (triggers default task creation)
- [x] **COMP-03**: Detail page Section 1 "Basisgegevens": company name, warmup start date, go-live date, client email, client phone
- [x] **COMP-04**: Detail page Section 2 "Klantprofiel": client notes textarea, personality scale 1-5 visual selector
- [x] **COMP-05**: Detail page Section 3 "Mailvarianten": 3 large text blocks side by side (Variant 1, 2, 3)
- [x] **COMP-06**: Detail page Section 4 "Feedback & Planning": feedback mailvarianten, toekomstige wensen, "Extra shit" textareas
- [x] **COMP-07**: Auto-save on blur with visible save indicator, plus manual save button as fallback
- [x] **COMP-08**: Back button to return to companies overview

### UI & Styling

- [x] **UIST-01**: Tailwind CSS styling with shadcn/ui components (buttons, modals, inputs, toggles)
- [x] **UIST-02**: Light mode: white backgrounds, subtle gray borders, clean typography
- [x] **UIST-03**: Dark mode: dark gray backgrounds with appropriate contrast
- [x] **UIST-04**: Task status colors: red for pending, green for completed, orange/yellow for urgent
- [x] **UIST-05**: Overdue badges: red background, white text, small pill shape
- [x] **UIST-06**: Dutch labels throughout (button text, headers, placeholders)
- [x] **UIST-07**: Responsive but desktop-first layout
